## Jira Statistics
### How to run
```
$ git clone https://github.com/salespaulo/jira-statistics.git
$ cd jira-statistics
$ npm install
$ node app.js
```
### How install Node.js
#### Linux
```
$ sudo apt-get install nodejs nodejs-dev npm
```

#### MacOS
```
$ brew install node
```
